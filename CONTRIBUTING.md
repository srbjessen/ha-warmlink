## Adding New Sensor Codes

1. Fork this repository
2. Add code to `custom_components/warmlink/codes.json`
3. Add translations to `translations/sensor_en.json` and `translations/sensor_da.json`
4. Add units to `units.json` (if applicable)
5. Add icons to `icons.json` (if applicable)
6. Test locally
7. Submit pull request

## Code Style

- Follow PEP 8
- Add docstrings
- Keep functions focused
- Test your changes

## Pull Request Process

1. Create a branch: `git checkout -b feature/your-feature`
2. Make changes
3. Commit: `git commit -m "Add: Description"`
4. Push: `git push origin feature/your-feature`
5. Create Pull Request on GitHub

## Reporting Bugs

Use our [bug report template](.github/ISSUE_TEMPLATE/bug_report.md).

Include:
- Home Assistant version
- Integration version
- Heat pump model
- Steps to reproduce
- Relevant logs

## Code of Conduct

Be respectful and constructive. We're all here to help each other!
